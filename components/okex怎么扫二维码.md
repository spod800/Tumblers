## 标准操作流程

### 一、环境准备
1. 确保账户完成实名认证
2. 更新至最新版本APP（v6.23.1以上）
3. 开启双重验证功能
4. 确保设备摄像头工作正常

### 二、扫码功能调用路径
```markdown
[资产中心] → [钱包] → [收付] → [扫码识别]
```

### 三、具体操作步骤
1. **对准扫描**：保持设备与二维码平面呈30-60度角，距离15-30cm
2. **参数确认**：
   - 自动识别的地址信息
   - 交易金额（精确到小数点后8位）
   - 网络手续费估算
3. **二次验证**：输入交易密码或指纹认证
4. **执行交易**：点击确认按钮完成操作

> ⚠️ 安全提示：切勿在公共场合扫描来源不明的二维码，建议开启地址白名单功能
